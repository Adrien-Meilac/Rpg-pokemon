Les programmes à exécuter se trouvent dans bin et se nomment 
rpg-pokemon (couleurs 1).exe
rpg-pokemon (couleurs 2).exe
Ils ont le même contenu, seul couleur des graphismes et le pokemon ennemi changent.

Le code est dans code et nous avons mis les fichiers de données dans le répertoire bin et dans le répertoire code pour que l'on puisse exécuter le code directement depuis CodeBlocks.

"code Python" contient les morceaux de codes qui nous ont été nécessaires mais qui ne font pas directement parti du projet.

Nous avons aussi mis une version de comparaison dans notre archive :
"vrai jeu pokemon" contient un émulateur ainsi qu'une ROM du jeu Pokémon Rouge Feu dont sont issues les captures d'écran avec lesquelles nous comparons notre interface. Pour le lancer, il faut :
exécuter l'émulateur VisualBasicAdvance.exe
Aller dans Fichier > Open et sélectionner Pokemon - Fire Red Version (U) (V1.1).gba
Aller dans Load et chercher le fichier Pokemon - Fire Red Version (U) (V1.1).sgm
